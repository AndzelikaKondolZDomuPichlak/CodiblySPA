# Products List Application

This project is a web application that displays a list of products using Vite, React, TypeScript, and Zustand for state management. Styling is accomplished with SCSS.

## Requirements

- Node.js version 20.11.0 or higher

## Installation

To get started with this project, clone the repository and install the dependencies:

git clone https://your-repository-link
cd your-repository-directory
npm install

### Development server

npm run start

This will start the development server and you can open http://localhost:5173 to view it in the browser.

### Build for production

npm run build

This builds the app for production to the `dist` folder.

## Usage

To filter the products by ID, use the "Filter by ID" input. You can also navigate through the product list using the pagination controls.
